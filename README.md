# 有機グラスフェッドビーフ 採算・販売価格シミュレーター

北海道の有機グラスフェッドビーフ生産者向けに、飼養コストや加工・販売コストを入力するだけで、**最低・推奨・ブランド販売単価**を自動算出するWEBアプリです。

---

## アプリ概要

このツールは単なる原価計算ではなく、「**少頭数・高品質・認証価値・環境配慮**」を前提に、安売りを避けながら採算を見える化するための価格設計ツールです。

- **最低販売価格**: 赤字を回避するための下限価格
- **推奨販売価格**: 継続経営のための現実的な価格
- **ブランド販売価格**: 高付加価値・少頭数育成を守るための価格

---

## 想定ユーザー

- 有機JASやグラスフェッド認証を持つ、またはそれに近い価値観の畜産農家
- 少頭数育成で品質と自然環境を重視している生産者
- 大量販売ではなく高付加価値販売を前提とした農家
- 直販・卸・ふるさと納税・イベント販売を使い分けたい生産者

---

## 機能一覧

- 飼養コスト・認証費用・加工販売費の入力
- 最低・推奨・ブランド価格の自動計算
- 1kg単価・100g単価の表示
- 心理価格への自動丸め
- 年間想定売上・利益の表示
- 販売チャネル別ヒント表示
- localStorage による入力値の自動保存・復元
- サンプルデータの自動入力
- バリデーション・警告表示

## 計算ロジック

### 1頭あたり飼養コスト
```
飼料 + 牧草 + 衛生獣医 + 管理 + 人件費 + その他
```

### 1頭あたり認証・ブランドコスト
```
年間認証コスト合計 ÷ 年間出荷頭数
```

### 1頭あたり総原価
```
飼養コスト + 認証コスト（頭割） + 加工販売固定コスト
```

### 最低販売価格（1頭）
```
(総原価 + 最低利益額) ÷ (1 - 手数料率)
```

### 推奨販売価格（1頭）
```
max(
  総原価 ÷ (1 - 手数料率 - 理想利益率),
  最低価格 × 1.1
)
```

### ブランド販売価格（1頭）
```
推奨価格 × 価格戦略係数
  控えめ: ×1.05 / 標準: ×1.15 / 高付加価値: ×1.30
```

### 心理価格帯

**100g単価候補**: 398・480・580・680・780・880・980・1,180・1,280・1,480円
**1kg単価候補**: 3,980・4,800・5,800・6,800・7,800・8,800・9,800・11,800・12,800・14,800円

最低価格を下回らない範囲で最も近い候補に丸めます。

## GitHub Pages 公開方法

1. リポジトリを GitHub に push する
2. `vite.config.ts` の `base` をリポジトリ名に合わせる（例: `/grassfed-beef-simulator/`）
3. `npm run build` を実行
4. GitHub Actions（`.github/workflows/deploy.yml`）を設定:

```yaml
name: Deploy to GitHub Pages
on:
  push:
    branches: [main]
jobs:
  deploy:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-node@v4
        with:
          node-version: 20
      - run: npm ci
      - run: npm run build
      - uses: peaceiris/actions-gh-pages@v4
        with:
          github_token: ${{ secrets.GITHUB_TOKEN }}
          publish_dir: ./dist
```
